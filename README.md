# Automatic Resume Generator

Haven't you been in a situation where a company is interested in your abilities and asks for your CV, but you have to make slight modifications to it in order to send it over?

Those days are over, with _automatic resume generator_!

## What is Automatic Resume Generator?

Automatic Resume Generator is a Free, Open Source, and Easy to use program that allows you to generate personalized CVs for each job posting that you like, while using your own skills, re-ordering them so they match the keywords and order of the job posting, getting you a better chance of your CV to NOT be in the trash after those first 7 seconds that hiring managers use to identify whether you are worth it or not.

Automatic Resume Generator only asks three things from you:

1. An API Key from your favorite A.I Service
2. An URL to the job you want
3. Your LinkedIn URL

And it will return one beautiful, harvard-templated, ATS friendly, curriculum vitae adjusted for the company and job posting you are applying for!

## Limitations

Right now, Automatic Resume Generator is thought for Software Engineers and alike, with plans to expand for Marketing and Analytics roles!

## How to install

I am using `uv`, the greatest package manager for Python. You just need to install `uv` [(see installation instructions here)](https://docs.astral.sh/uv/getting-started/installation/) and then run:

`uv install`
`uv run main.py`
